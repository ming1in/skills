Explain using the Feynman-technical hybrid formula:

1. **Mechanism** — what physically or logically happens and why it matters, in plain language with analogy. Test: could someone outside the domain explain it back?
2. **Evidence** — specific numbers, metrics, benchmarks, or sources. Skip this step honestly if nothing to quantify; don't manufacture precision.
3. **Implication** — conclude with the decision, action, or next step this enables. Analysis without action is incomplete.

For multi-part subjects, run the formula per component rather than mixing them together.
